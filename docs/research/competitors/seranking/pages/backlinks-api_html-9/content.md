FAQ
Q
What data does the Backlinks API include?
A

The Backlinks API gives you loads of useful data, including live backlinks, anchor text, dofollow/nofollow status, InLink Rank scores, link types, referring domains, IPs, subnets, TLDs, and more. Track new and lost links, check backlink history, and get deep data on each referring site’s quality and trustworthiness.

Q
Can I filter by anchor text, TLD, or link type?
A

Yes, you can filter by all of those. Use the SEO Backlink API to narrow down results by anchor text patterns, specific TLDs (like .edu or .gov), and link types (direct links vs redirects). Filter by dofollow/nofollow status, date ranges, and more, to focus on the exact data you need.

Q
Does it support historical tracking and delta views?
A

Yes, it does! The Backlinks API tracks backlink and referring domain changes over any date range and reveals what’s new or lost each day. The history endpoints show you when links appeared or disappeared, why they were lost (like page not found or link removed), and which referring domains started or stopped linking to you.

Q
Can I bulk-check multiple domains in one request?
A

Yes, you can! The Backlink API supports bulk checks. Send a list of up to 100 domains in a single request using the summary endpoint. Note that detailed endpoints like history, all, and raw backlinks only handle one domain per request.

Q
How frequently is backlink data updated?
A

SE Ranking delivers some of the freshest backlink data available, with 58% of backlinks updated every 90 days. It also detects new backlinks fast to give you the latest snapshot of your backlink profile.

Q
What is the API rate limit for the Backlinks API?
A

You’re allowed 10 requests per second per API key. Going over this triggers a “Too Many Requests” response. Pause briefly and retry to minimize hiccups. Every successful request consumes API units from your quota, so keep an eye on your balance. If you need more units, reach out to SE Ranking.

Q
Can I export complete backlink datasets without hitting rate limits?
A

Absolutely. For getting large-scale backlink datasets, use the asynchronous export endpoint. Queue your request, get a task ID back, then poll the status until it’s complete. Once ready, the export comes in a compact CSV format, perfect for large-scale SEO data analysis without hitting rate limits.

Q
Can I get the full list of backlinks and referring domains via API?
A

Yes, absolutely. This applies to any target. Use the export endpoint to download the full backlink dataset as a compressed CSV file. Or stream up to 100,000 backlinks at a time through the raw endpoint to work with smaller chunks. The same applies to referring domains. Use the refdomains and refdomains/count endpoints to get a complete list or just the totals, with full detail included.
